## Quick Start

```typescript
import { geoai } from "@geobase-js/geoai";

// Initialize pipeline
const pipeline = await geoai.pipeline(
  [{ task: "car-detection" }],
  providerParams
);

// Run detection
const result = await pipeline.inference({
  inputs: { polygon: myPolygon },
});

console.log(`Found ${result.detections.features.length} vehicles`);
```
